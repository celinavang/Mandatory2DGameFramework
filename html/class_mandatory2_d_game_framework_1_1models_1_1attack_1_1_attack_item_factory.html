Detailed Description

Factory class for creating attack items.

Provides a static method for creating instances of IAttackItem based on the specified attack type. Uses the AttackType enumeration to determine the type of item to create (melee or ranged).

static

Creates an attack item of a specified type.

This method creates and returns a melee or ranged attack item based on the provided AttackType. Logs the creation of each item type and throws an exception if an invalid attack type is provided.

Parameters
attackTypeThe type of attack item to create. This can be either AttackType.melee or AttackType.ranged.
nameThe name of the attack item.
maxDamageThe maximum damage value for the attack item.
Returns
An instance of IAttackItem corresponding to the specified attack type.
Exceptions
ExceptionIf an invalid attack type is provided.
Note
This factory method uses tracing to log information about the item creation.
