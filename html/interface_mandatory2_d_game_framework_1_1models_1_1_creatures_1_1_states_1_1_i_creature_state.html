World world,
IWorldObject obj )

Handles the state of the creature. The method will be called when the creature is in the state and it is the creatures turn. The method will return a boolean value, which will determine if the creature has performed an action on its turn.

Parameters
creatureA reference to the creature that is in the state. This is the creature whose behavior is being handled.
worldThe world in which the creature and the object exist. This allows checking for collisions and interactions with the environment.
objThe object that the creature is interacting with. This can be another creature or an item.
