\doxysubsection{Detailed Description}
The \doxylink{class_mandatory2_d_game_framework_1_1models_1_1_creatures_1_1_states_1_1_move_state}{Move\+State} class handles the behavior of a creature when it is in a state of moving towards an object. 

The state is activated when the creature begins its turn. The movement is calculated based on the relative position of the object, and the creature will either move towards the object or transition to another state (e.\+g., attacking) if it can interact with the object.
